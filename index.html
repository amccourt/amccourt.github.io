UMD Schedule of Classes Extension

  • Description
  • Github
  • Chrome Web Store
Created With:
  • Javascript
  • JQuery

This is a chrome extension that I created to solve an issue that I had with my school's Schedule of Classes.

Picking classes for the semester was difficult because I would have to look up professor ratings for each available section manually.

This extension adds color-coded scores on a 0-5 scale pulled from RateMyProfessor.com. Each score is displayed next to the professors name to allow section choices to be made easier for students.

NetBinge - A Netflix Chrome Enhancement

  • Description
  • Github
  • Chrome Web Store
Created With:
  • Javascript

I created this extension to add some new features to improve Netflix viewer experience, as well as to practice using Javascript for Chrome development.

I watch Netflix from my bed often and don't like getting up to use my mouse to skip intro sequences or remove the annoying "Are you still watching?" message.

This extension automates these features, and also blurs the episode descriptions on Netflix which often contain spoilers. Each feature is easily toggle able in a simple popup menu that I created.
